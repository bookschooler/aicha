# 서울 찻집 블루오션 상권 분석 — PPT 제작 요청

## 전체 디자인 방향

### 스타일
- BCG / McKinsey 컨설팅 덱 스타일: 슬라이드마다 단 하나의 핵심 메시지(header message)를 가장 크게 배치하고, 아래에 근거와 시각 자료를 구성
- 군더더기 없는 레이아웃: 텍스트는 최소화, 데이터·도표가 중심
- 각 슬라이드 좌측 상단 또는 상단 바에 슬라이드 번호 + 섹션명 표시

### 컬러 팔레트
- 배경: 짙은 다크 그린 (#1B3A2D 또는 유사 계열)
- 주요 텍스트: 화이트 (#FFFFFF)
- 포인트 컬러: 골드 (#C9A84C 또는 유사 계열) — 핵심 수치, 강조 박스, 구분선에 사용
- 보조 컬러: 밝은 그린 (#4CAF7D) — 긍정 지표, Q1 강조
- 차트 내 사분면 색상: Q1=밝은 골드, Q2=밝은 그린, Q3·Q4=회색 계열

### 타이포그래피
- 제목(Header Message): 28~34pt, Bold, 화이트 또는 골드
- 소제목 / 라벨: 16~18pt, SemiBold, 화이트
- 본문 / 표 내용: 13~14pt, Regular, 연한 화이트 (#E0E0E0)
- 영문 폰트: Inter 또는 Montserrat / 한글 폰트: Noto Sans KR 또는 Pretendard

### 레이아웃 & 여백
- 슬라이드 사이즈: 16:9 와이드 (1920×1080 기준)
- 좌우 여백: 80~100px, 상하 여백: 60~80px
- 컨텐츠 영역과 여백을 명확히 구분하여 답답하지 않게 유지
- 표·차트는 슬라이드 면적의 최대 70% 이내로 제한
- 섹션 구분선: 골드 컬러 1~2px 가로선

### S6. Track B — 복합 공급부족 지수
**Header Message:** "찻집이 없고, 경쟁 속에서도 기회가 있는 곳을 정량화했다"
**구성 요소:**
- 두 요소를 나란히 카드 형태로 배치:
  - 카드 1 (골드): "1/(찻집수+1)" → "찻집이 0개일수록 만점"
  - 카드 2 (그린): "찻집매출/(카페점포수+1)" → "경쟁 속 찻집 성과 반영"
- 두 카드 아래: "rank 변환 후 0.5/0.5 가중합 → 두 차원 동등 고려"
- 수식 박스 (골드 테두리): 공급점수 = 0.5×rank(1/(찻집수+1)) + 0.5×rank(찻집매출/(카페점포수+1))

**이미지 요청:** 불필요
