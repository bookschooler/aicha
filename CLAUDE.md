## 핵심 방법론: 2-Track 프레임워크

```
Track A: 순수 수요 예측 모델 (Pooled OLS)
  X = 수요지표 6개 + 분기더미 8개 + 상권유형더미
  Y = log(찻집업종 당월 매출)
  → GroupKFold(n_splits=5, groups=상권_코드) OOF 잔차 추출
     잔차 양수 = 수요 대비 과성과 (이미 잘 됨)
     잔차 음수 = 수요 대비 저성과 (미실현 잠재력)

Track B: 복합 공급부족 지수 (모델 밖에서 계산)
  = 0.5 × rank(1/(찻집수+1))
  + 0.5 × rank(찻집매출/(카페음료점포수+1))

→ 2D 매트릭스 사분면:
  Q1 (잔차≥0 + 찻집없음) = 검증시장공백 → 안전 진입 전략
  Q2 (잔차<0  + 찻집없음) = 잠재수요미실현 → 선점 전략
```

**왜 공급변수를 Track A에서 제외했나?**
- Y(매출)는 공급이 있어야 발생 → 공급변수 포함 시 잔차 해석 붕괴
- 잔차가 "찻집 적음"을 이미 포착하면 Track B가 중복됨
- Track A = "커피·음료 시장이 얼마나 실현됐나", Track B = "찻집이라는 니치가 비어있나"
