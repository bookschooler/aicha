# 서울 찻집 블루오션 상권 분석 — 최종 보고서

> **작성일:** 2026-03-06  
> **목적:** 서울시 찻집 창업을 위한 블루오션 상권 데이터 기반 추천  
> **분석 기간:** 2023Q3 ~ 2025Q3 (9분기)

---

## 1. 프로젝트 개요

단순히 매출이 높은 상권이 아니라, **"수요는 충분한데 찻집이 없는 상권"** 을 발굴하는 것이 목표.

```
데이터:  서울시 1,139개 상권 × 9분기 패널 (원본 9,760행 / 155개 변수)
목표:    찻집 창업 최적 상권 추천 (블루오션 = 수요 있음 + 찻집 없음)
```

---

## 2. 분석 방법론 — 2-Track 프레임워크

```
┌─────────────────────────────────────────────────┐
│  Track A: 순수 수요 예측 모델 (Pooled OLS)       │
│    X = 수요지표 6개 + 분기더미 + 유형더미         │
│    Y = log(찻집업종 당월 매출)                    │
│    → GroupKFold OOF 잔차 추출                    │
│       잔차 양수 = 수요 대비 과성과 (이미 잘 됨)   │
│       잔차 음수 = 수요 대비 저성과 (아직 미실현)  │
└─────────────────────────────────────────────────┘
                      +
┌─────────────────────────────────────────────────┐
│  Track B: 복합 공급부족 지수 (모델 밖)           │
│    = 0.5 × rank(1/(찻집수+1))                   │
│    + 0.5 × rank(찻집매출/(카페음료점포수+1))     │
└─────────────────────────────────────────────────┘
                      ↓
┌─────────────────────────────────────────────────┐
│  2D 매트릭스 → 사분면 분류                       │
│    Q1 (파랑): 잔차>=0 + 찻집없음 = 검증시장공백  │
│    Q2 (빨강): 잔차<0  + 찻집없음 = 잠재수요미실현│
└─────────────────────────────────────────────────┘
```

**수요 변수 6개 (XGBoost SHAP → Lasso → VIF 최종 확정)**

| 변수 | SHAP 순위 | VIF |
|------|---------|-----|
| 집객시설_수 | 2위 (0.401) | 2.89 |
| 총_직장_인구_수 | 3위 (0.265) | 1.72 |
| 월_평균_소득_금액 | 4위 (0.260) | 3.28 |
| 총_가구_수 | 7위 (0.136) | 2.37 |
| 카페_검색지수 | 9위 (0.104) | 1.62 |
| 지하철_노선_수 | 17위 (0.031) | 1.74 |

## 3. 모델 성능

| 구분 | R² |
|------|-----|
| **OOF R² (GroupKFold 5-fold)** | **0.4413** |
| 전체 OLS R² (in-sample) | 0.4581 |

> OOF R²=0.44는 공급변수를 의도적으로 제외한 수요 전용 모델 기준으로 정상.  
> R²이 너무 높으면 잔차가 0에 수렴해 블루오션 신호 소멸. 현재 수준이 최적.

## 6. 전략 제언

### 리스크 성향별 선택

| 성향 | 추천 전략 | 대표 상권 |
|------|---------|---------|
| 안전 추구 | **Q1** — 검증된 수요 + 공급 공백 | 홍대소상공인상점가, 외대앞역 1번 |
| 선점·성장 | **Q2** — 잠재수요 미실현 선점 | 서울역, 대치역 |
| 균형 | Q1 Top3 + Q2 Top2 조합 검토 | — |

### Q1 상권 특성 요약

- 주로 **골목상권·전통시장** (홍대, 외대, 성신여대 등 대학가 인근)
- 잔차 +2.5~+3.2 — 같은 수요 조건 상권 대비 매출이 현저히 높음
- 집객시설·가구수 지표가 강점 → 상주 소비층 탄탄

### Q2 상권 특성 요약

- 주로 **발달상권** (서울역, 대치역, 상봉역 등 교통 허브)
- 직장인구·집객시설이 전국 상위 3~7% — 압도적 수요 기반
- 잔차 -0.1~-1.0 — 수요 대비 찻집 매출이 낮음 = 미실현 잠재력

### 주의사항

- 본 분석은 **상권 단위** 추천. 실제 점포 위치(도로, 건물 유형, 임대료)는 현장 조사 필요.
- OOF R²=0.44 → 수요 외 설명 안 되는 56% 분산 존재. 잔차에 노이즈 포함 가능성.
- 분석 기준 분기: 2025Q3. 이후 상권 변화(대규모 개발, 찻집 신규 개업 등) 모니터링 필요.

## 7. 공급지수 가중치 민감도 분석

> **배경:** 복합 공급지수의 두 요소(찻집희소성 / 점포당매출)에 동일 가중치(0.5/0.5)를 적용한 것에 대한 방어 근거 확보.
> **방법:** 가중치를 0.1/0.9 ~ 0.9/0.1 까지 9단계로 변화시키며 Q1/Q2 Top5 일치율 측정.

### 분석 결과

| 가중치 (찻집희소성/점포당매출) | Q1 Top5 일치율 | Q2 Top5 일치율 |
|--------------------------|:------------:|:------------:|
| 0.1 / 0.9 | 3/5 | 3/5 |
| 0.2 / 0.8 | 3/5 | 3/5 |
| 0.3 / 0.7 | 3/5 | 3/5 |
| 0.4 / 0.6 | 4/5 | 4/5 |
| **0.5 / 0.5 (기준)** | **5/5** | **5/5** |
| 0.6 / 0.4 | 5/5 | 4/5 |
| 0.7 / 0.3 | 5/5 | 4/5 |
| 0.8 / 0.2 | 4/5 | 3/5 |
| 0.9 / 0.1 | 4/5 | 1/5 |
| **평균** | **4.0/5** | **3.3/5** |

### 해석

- **Q1 (안전 진입 전략): Robust** — 가중치가 어떻게 바뀌어도 추천 상권이 안정적으로 유지됨.
- **Q2 (선점 전략): 합리적 범위에서 안정** — 두 지표를 각각 30~70% 배분하는 범위(w=0.3~0.7)에서 4~5/5 일치. 한 지표에 90% 이상 치우치는 극단적 가중치에서는 변동 발생.
- **결론:** 0.5/0.5는 두 지표를 동등하게 고려하는 중립적·합리적 기본값이며, 이 선택이 결과의 안정성을 해치지 않음.

> 참고: 두 지표를 "복합 활용"하는 설계 취지상, 한 지표가 90%를 차지하는 가중치(0.9/0.1)는 설계 의도에 반함.
